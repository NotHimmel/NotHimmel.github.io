添加表这个DDL操作，只会在数据库中的目录(Catalog)中新增一个条目，并返回一个状态如：
```psql
jas=# create table RR ( x integer, y integer);
CREATE TABLE

jas=# \d rr
    Table "Public.rr"
Column | Type   | Modifiers
-------+---------+-----------
x      | integer |
y      | integer |      
```
